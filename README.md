# csswind

Live site: [csswind.com](https://www.csswind.com)

A terminal-styled quiz app for learning CSS and Tailwind. Each round gives you 20 questions, mixing Tailwind class to CSS property and CSS property to Tailwind class. You have 4 minutes to answer as many as you can from memory.

## How it works

- **Submit** with Enter. If correct, move on. If wrong, try again.
- **Pass** with Esc. The correct answer is shown before moving on.
- Answers are checked leniently: case, spaces around `:` and `/`, and trailing semicolons are all ignored.

## Stack

- [React](https://react.dev)
- [Vite](https://vitejs.dev)

## Getting started

```bash
npm install
npm run dev
```
